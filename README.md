# giraphant/homebrew-tap

Homebrew tap for [Dousha](https://github.com/giraphant/dousha), a menu-bar push-to-talk dictation app for macOS.

## Install

```sh
brew install --cask giraphant/tap/dousha
```

On first launch, grant Microphone, Speech Recognition, and Accessibility permissions when prompted.

## Update

```sh
brew upgrade --cask dousha
```

## Uninstall

```sh
brew uninstall --cask dousha          # remove the app
brew uninstall --zap --cask dousha    # also remove preferences, caches, credentials
```
